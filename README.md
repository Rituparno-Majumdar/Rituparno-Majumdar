## Bengali AI Annotation

I work on Bengali language AI training data — annotation, evaluation, and quality assessment for low-resource Indian language NLP.

| Repo | What it does |
|---|---|
| 🔬 [`bengali-annotation-quality`](https://github.com/Rituparno-Majumdar/bengali-annotation-quality) | Gold-standard calibration: blind re-annotation of BLP-2023 Bengali sentiment dataset with Cohen's Kappa measurement |
| 🗣️ [`bengali-annotation-job-alert`](https://github.com/Rituparno-Majumdar/bengali-annotation-job-alert) | Monitors 11 sources every 6 hours for Bengali NLP annotation roles — Python + GitHub Actions + Telegram |
| 📊 `bengali-codemix-dataset` | Labelled Bengali-English code-mixed sentences with naturalness ratings and code-switch boundary tags *(coming soon)* |
| 🏷️ `bengali-sentiment-annotation` | Fine-grained Bengali sentiment and hate speech annotations with cultural edge case documentation *(coming soon)* |
| 🔤 `bengali-ner-annotation` | Bengali Named Entity Recognition dataset with annotation guidelines *(coming soon)* |
| 🐍 `bengali-nlp-python` | Python notebooks: Unicode normalisation, HuggingFace Bengali datasets, BNLP tokenisation *(coming soon)* |
| ⚖️ `bengali-llm-evaluation` | RLHF preference ranking · MQM translation quality · Bengali safety annotation *(coming soon)* |

**Why Bengali?** Bengali is the 7th most spoken language in the world and one of the most underrepresented in LLM training data. Native fluency combined with structured annotation methodology is a rare and necessary combination for building AI systems that actually work in Bengali.
